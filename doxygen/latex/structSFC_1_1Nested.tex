\doxysubsection{Detailed Description}
Implementation of a nested index SFC. 

The SFC is encoded by the code = i + j \texorpdfstring{$\ast$}{*} N + k \texorpdfstring{$\ast$}{*} N \texorpdfstring{$\ast$}{*} N in 3D, where i,j,k are the block indices.
