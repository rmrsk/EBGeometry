\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classVec3T}{class}} \mbox{\hyperlink{classVec3T}{T}}$>$\newline
class Signed\+Distance\+Function$<$ T $>$}
Abstract representation of a signed distance function. 

Users can put whatever they like in here, \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{e.\+g}. analytic functions, \doxylink{namespaceDCEL}{DCEL} meshes, or \doxylink{namespaceDCEL}{DCEL} meshes stored in full or compact \doxylink{namespaceBVH}{BVH} trees. The signed\+Distance function must be implemented by the user. When computing it, the user can apply transformation operators (rotations, scaling, translations) by calling transform\+Point on the input coordinate.
