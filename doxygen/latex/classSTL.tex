\subsubsection*{template$<$\mbox{\hyperlink{classVec3T}{typename}} \mbox{\hyperlink{classVec3T}{T}}$>$\newline
class STL$<$ T $>$}
Class for storing contents of \doxylink{classSTL}{STL} files. 

This structure is simply used as conversion utility between \doxylink{classSTL}{STL} files and various \doxylink{namespaceEBGeometry}{EBGeometry} objects (triangle soups, \doxylink{namespaceDCEL}{DCEL} meshes, etc). \begin{DoxyNote}{Note}
T is the precision used for storing the mesh. 
\end{DoxyNote}
