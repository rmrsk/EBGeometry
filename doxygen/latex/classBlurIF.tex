class Blur\+IF$<$ T $>$}
Blurred/interpolated implicit function -\/ can be used for smoothing. 

This passed the input implicit function through a trilinear filter with specified distance. We blur the function using f = alpha \texorpdfstring{$\ast$}{*} f + (1-\/alpha)/2 \texorpdfstring{$\ast$}{*} \mbox{[}f(x+d) + f(x-\/d)\mbox{]}
