\begin{DoxyNote}{Note}
When using parallel calculations it is exceptionally important that the input RNG is the same across all threads/ranks. Otherwise, the user must manually ensure that the permutation table is the same. Failure to do so implies that each thread/rank generates it\textquotesingle{}s own gradient noise and there is correspondingly no single geometry.
