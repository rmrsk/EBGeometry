\doxysubsection{Detailed Description}
File which contains partitioners and lambdas for enclosing \mbox{\hyperlink{namespaceDCEL}{DCEL}} faces in bounding volume hierarchies. 

This file contains various useful \char`\"{}default\char`\"{} routines for determining how a \mbox{\hyperlink{namespaceDCEL}{DCEL}} mesh should be partitioned in a bounding volume hierarchy.
