\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T, class P, class BV, size\+\_\+t K$>$\newline
class BVH\+::\+Linear\+BVH$<$ T, P, BV, K $>$}

Forward declare linear \mbox{\hyperlink{namespaceBVH}{BVH}} class. 

Linear root node for \mbox{\hyperlink{namespaceBVH}{BVH}} hierarchy.

T is the precision used in the \mbox{\hyperlink{namespaceBVH}{BVH}} computations, P is the enclosing primitive and BV is the bounding volume used in the \mbox{\hyperlink{namespaceBVH}{BVH}}. K is the tree degree.
