\subsubsection*{template$<$\mbox{\hyperlink{classVec3T}{class}} \mbox{\hyperlink{classVec3T}{T}}, \mbox{\hyperlink{classVec3T}{class}} \mbox{\hyperlink{classVec3T}{Meta}}$>$\newline
class DCEL\+::\+Vertex\+T$<$ T, Meta $>$}
Vertex class. 

Class which represents a vertex node in a double-\/edge connected list (\doxylink{namespaceDCEL}{DCEL}).

This class is used in \doxylink{namespaceDCEL}{DCEL} functionality which stores polygonal surfaces in a mesh. The \doxylink{classDCEL_1_1VertexT}{VertexT} class has a position, a normal vector, and a pointer to one of the outgoing edges from the vertex. For performance reasons we also include pointers to all the polygon faces that share this vertex. \begin{DoxyNote}{Note}
The normal vector is outgoing, \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{i.\+e}. a point x is "{}outside"{} the vertex if the dot product between n and (x -\/ x0) is positive. 
\end{DoxyNote}
