class Union\+B\+V\+H$<$ T, B\+V, K $>$}

Distance function union using B\+V\+Hs. Computes the signed distance to the closest object of N non-\/overlapping objects. 

\begin{DoxyNote}{Note}
This class only makes sense if the object do not overlap.
