class Finite\+Repetition\+IF$<$ T $>$}
Class which creates a periodic repetition of an implicit function. 

The user will specify the period (\doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{i.\+e}., spacing), and the number of repetitions along increasing and decreasing coordinate directions.
