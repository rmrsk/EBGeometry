\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classVec3T}{class}} \mbox{\hyperlink{classVec3T}{T}}, \mbox{\hyperlink{classVec3T}{class}} \mbox{\hyperlink{classVec3T}{Meta}}$>$\newline
class DCEL\+::\+Edge\+T$<$ T, Meta $>$}
Edge class. 

Class which represents a half-\/edge in a double-\/edge connected list (\doxylink{namespaceDCEL}{DCEL}).

This class is used in \doxylink{namespaceDCEL}{DCEL} functionality which stores polygonal surfaces in a mesh. The information contain in an \doxylink{classDCEL_1_1EdgeT}{EdgeT} object contains the necessary object for logically circulating the inside of a polygon face. This means that a polygon face has a double-\/connected list of half-\/edges which circulate the interior of the face. The \doxylink{classDCEL_1_1EdgeT}{EdgeT} object is such a half-\/edge; it represents the outgoing half-\/edge from a vertex, located such that it can be logically represented as a half edge on the "{}inside"{} of a polygon face. It contains pointers to the polygon face, vertex, and next half edge It also contains a pointer to the "{}pair"{} half edge, \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{i.\+e}. the corresponding half-\/edge on the other face that shares this edge. Since this class is used with \doxylink{namespaceDCEL}{DCEL} functionality and signed distance fields, this class also has a signed distance function and thus a "{}normal vector"{}. \begin{DoxyNote}{Note}
The normal vector is outgoing, \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{i.\+e}. a point x is "{}outside"{} if the dot product between n and (x -\/ x0) is positive. 
\end{DoxyNote}
