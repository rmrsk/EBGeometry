Compute an approximation to the bounding volume for the implicit surface, using octrees. 

This routine will try to compute a bonding using octree subdivision of the implicit function. This routine will characterize a cubic region in space as being \textquotesingle{}inside\textquotesingle{}, \textquotesingle{}outside\textquotesingle{}, or intersected by computing the value function at the center of the cube. If the value function is larger than the extents of the cube, we assume that there is no intersection inside the cube. The success of this algorithm therefore relies on the implicit function also being a signed distance function, or at the very least not being horrendously far from being an SDF. If octree subdivision fails, this will return the maximally representable bounding volume.
