\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Implicit\+Function$<$ T $>$}

Abstract representation of an implicit function function (not necessarily signed distance). 

The value function must be implemented by subclasses. When computing it, the user can apply transformation operators (rotations, scaling, translations) by calling transform\+Point on the input coordinate.
