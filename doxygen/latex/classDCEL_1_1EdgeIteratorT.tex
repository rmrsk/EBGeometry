\subsubsection*{template$<$class T$>$\newline
class D\+C\+E\+L\+::\+Edge\+Iterator\+T$<$ T $>$}

Class which can iterate through edges and vertices around a \mbox{\hyperlink{namespaceDCEL}{D\+C\+EL}} polygon face. 

This class can be used so that it either visits all the half-\/edges in a face, or all the outgoing half-\/edges from a vertex.
