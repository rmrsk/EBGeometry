\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Vec3\+T$<$ T $>$}
Three-\/dimensional vector class with arithmetic operators. 

The class has a public-\/only interface. To change it\textquotesingle{}s components one can call the member functions, or set components directly, \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{e.\+g}. vec.\+x = \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{5.\+0} \begin{DoxyNote}{Note}
\doxylink{classVec3T}{Vec3T} is a templated class primarily used with \doxylink{namespaceDCEL}{DCEL} grids. It is always 3D, \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{i.\+e}. independent of Chombo configuration settings. This lets one use \doxylink{namespaceDCEL}{DCEL} functionality even though the simulation might only be 2D. 
\end{DoxyNote}
