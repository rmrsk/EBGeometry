class Octree\+::\+Node$<$ Meta, Data $>$}
\doxylink{namespaceOctree}{Octree} class without anything special (this uses full tree representation rather than linear/pointerless). 

Meta is the meta-\/data for the node (user can put in \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{e.\+g}. level, volume, etc) and Data is the leaf node contents. \begin{DoxyNote}{Note}
This class should only be used with smart pointers. \doxylink{classVec3T_a314b7c2fe3a793f2e307a0093050f426}{I.\+e}. std\+::shared\+\_\+ptr$<$\+Node$>$ rather than \doxylink{classOctree_1_1Node}{Node}.
