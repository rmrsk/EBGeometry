\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Meta, typename Data = void$>$\newline
class Octree\+::\+Node$<$ Meta, Data $>$}

\mbox{\hyperlink{namespaceOctree}{Octree}} class without anything special (this uses full tree representation rather than linear/pointerless). 

Meta is the meta-\/data for the node (user can put in e.\+g. level, volume, etc) and Data is the leaf node contents. \begin{DoxyNote}{Note}
This class should only be used with smart pointers. I.\+e. std\+::shared\+\_\+ptr$<$\+Node$>$ rather than \mbox{\hyperlink{classOctree_1_1Node}{Node}}. 
\end{DoxyNote}
