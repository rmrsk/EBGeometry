Detailed Description

File which contains partitioners and lambdas for enclosing DCEL faces in bounding volume heirarchies.

This file contains various useful "default" routines for determining how a DCEL mesh should be partitioned in a bounding volume hierarchy. This includes the required functions for 1) Constructing bounding volumes (defaultBVConstructor). 2) Stopping the sub-division process (defaultStopFunction) 3) Partitioning one bounding volume into subvolumes.
