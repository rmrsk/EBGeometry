Detailed Description

template<class T, class P, class BV, size_t K>
class BVH::LinearBVH< T, P, BV, K >

Forward declare linear BVH class.

Linear root node for BVH hierarchy.

T is the precision used in the BVH computations, P is the enclosing primitive and BV is the bounding volume used in the BVH. K is the tree degree.

Stack-based pruning algorithm (recursion-less).

This will iterate through the BVH and visit all nodes where a_pruner evaluates to true. If the node is a leaf node, we update the "distance", which will be the closest object for signed distance, or the one with the smallest value for the constructive solid geometry union.

Parameters
[in]a_point3D point in space
[in]a_comparatorComparator for how to select the "output" when comparing various distances.
